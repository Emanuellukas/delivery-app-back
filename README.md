# Delivery App Back

API for a food delivery app built with node.js

## Setup

```bash
yarn
```

or

```bash
npm i
```

## Running

```
yarn start
```

or


```
npm run start
```

## Stack

- Database: Postgres
- ORM: Sequelize
- Authentication: jwt
- Encrypt: bcryptjs

## API Routes

Please import `Delivery-App-Insomnia.json` into Insomnia App to check out/use api routes/payloads
